How to Create the HR Portal Using React.js

This guide explains how to build an HR portal using React.js, Firebase, and Context API.

Prerequisites

  • Node.js and npm: Install from nodejs.org.
  • React.js: Basic knowledge required.
  • Firebase Account: Sign up at Firebase Console.

Project Setup

  1. Create React App 
    npx create-react-app hr-portal
cd hr-portal
  2. Install Dependencies 
    npm install react-router-dom axios bootstrap firebase
  3. Firebase Setup: Create and configure your Firebase project.

Implementation

  • Components: Develop reusable components like Header, Footer, Sidebar.
  • Routing: Use `react-router-dom` for routing.
  • State Management: Use Context API for global state management.
  • Firebase Integration: Set up Firebase Firestore to handle data operations.
  • Styling: Apply Bootstrap and custom CSS for styling.

Testing and Deployment

  • Testing: Write tests using Jest and React Testing Library.
  • Deployment: Build and deploy the app to platforms like Vercel or Netlify.

Understanding React, Frameworks, Context API, and Firebase

What is React?

React is a JavaScript library developed by Facebook for building user interfaces, particularly single-page applications where data changes over time. It allows for the creation of reusable UI components that manage their own state.

Official Site: React.js

What is a Framework?

A framework provides a foundation for developing software applications. It includes pre-written code and tools to streamline development. Examples include Angular, Vue.js, and Express.js.

React Context API

The Context API in React is a way to manage and share state globally without passing props manually through multiple levels of components.

Documentation: React Context API

How to Use Context API:

Create a Context:
import React, { createContext, useState } from 'react';

export const MyContext = createContext();

export const MyProvider = ({ children }) => {
    const [state, setState] = useState('default value');

    return (
        
            {children}
        
    );
};
Use Context in Components:
import React, { useContext } from 'react';
import { MyContext } from './MyProvider';

const MyComponent = () => {
    const [state, setState] = useContext(MyContext);

    return 
{state}
;
};

Firebase Overview

Firebase is a platform developed by Google that provides a suite of cloud-based tools for developing mobile and web applications. It offers features like real-time databases, authentication, and hosting.

Official Site: Firebase

How to Use Firebase:

Setup Firebase:
  1. Create a Firebase project in the Firebase Console.
  2. Add Firebase SDK to your project by installing it: 
    npm install firebase
  3. Initialize Firebase in your application: 
    import firebase from 'firebase/app';
import 'firebase/firestore';

const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_AUTH_DOMAIN",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_STORAGE_BUCKET",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};

firebase.initializeApp(firebaseConfig);

const db = firebase.firestore();
Using Firebase Firestore:

Firestore is a flexible, scalable database for mobile, web, and server development. You can perform CRUD operations as follows:

// Add a new document
db.collection('employees').add({
    name: 'John Doe',
    position: 'Developer',
    department: 'Engineering'
})
.then(() => console.log('Document successfully written!'))
.catch((error) => console.error('Error writing document: ', error));

// Retrieve documents
db.collection('employees').get()
    .then((querySnapshot) => {
        querySnapshot.forEach((doc) => {
            console.log(doc.id, ' => ', doc.data());
        });
    })
    .catch((error) => console.error('Error getting documents: ', error));

// Update a document
db.collection('employees').doc('DOCUMENT_ID').update({
    position: 'Senior Developer'
})
.then(() => console.log('Document successfully updated!'))
.catch((error) => console.error('Error updating document: ', error));

// Delete a document
db.collection('employees').doc('DOCUMENT_ID').delete()
    .then(() => console.log('Document successfully deleted!'))
    .catch((error) => console.error('Error deleting document: ', error));
